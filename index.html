🚀 What's New in v2.6.5 Tank?

🧊 Absolute Zero Bloat Policy

No Browser. No Snap. No Flatpak. No Telemetry. Gamma Linux ships as a pure skeleton. You choose Firefox, Brave, Lynx, or nothing. The user is 100% free from day 1.

⚙️ Universal Boot Engine v2

Full BIOS Legacy + UEFI + IA32 support. Dynamic kexec auto-detect injects safe kernel params for old Intel, AMD, Nvidia GPUs. No GRUB> rescue.

🛡️ Anti-Fragile Build System

Optimized 3-Edition Matrix. Built cleanly without broken or legacy-locked package dependencies like task-standard, ensuring deterministic ISO builds.

👻 Ghost Mode RAM

Per-Edition ZRAM Tuning. Legacy-32 uses an aggressive 200% ZRAM + Swappiness 100. Pro-MaxMini leverages EarlyOOM + smart cache pressure handling.

🖥️ Openbox + Enlightenment

Ultra-optimized desktop environments featuring Materia-Dark + Papirus. Compositing tuned to push idle RAM footprint down to the absolute absolute minimum.

🗽 User Freedom First

No Forced Browser: Firefox, Brave, Chromium, Lynx... You decide. We ship pure desktop environments. You add only what YOU need. That's real freedom.

🛠️ Installation

⚠️ CRITICAL: Disable Windows Fast Startup

Run powercfg /h off as Administrator in Windows before dual-boot.

1. Flash USB

Use Ventoy or Rufus (DD mode recommended for legacy HP laptops).

2. Boot Live

F12/F11/Esc -> USB -> Login: gamma/gamma

3. Install

Click the custom installer launcher on the desktop environment. Done.
