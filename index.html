🛠️ Installation

⚠️ CRITICAL: Disable Windows Fast Startup- Read or lose Windows !

Run powercfg /h off and disable "Fast startup" option in battery settings (Serach in Google) as Admin, or your disk will be read-only.

1. Burn USB

Use Ventoy or Rufus for windows users or "sudo dd" if you are a professional linux user. Copy ISO. Boot anywhere.

2. Boot Live (Recommend to test firstly)

F12 -> USB -> Login: gamma/gamma

3. Install

Click "Install Gamma". 5 minutes.

NOTE : If "Install Gamma" does not appear in Desktop, Shutdown and boot from USB again and choose "Install Gamma"in boot menu
